Two-Column Layout Using Floats & Grid

Left Column

This is the left column. It uses 6 columns out of a 12-column grid system, which means it takes up 50% of the container width. Floats are used to position this column beside the right one.

You can add any content here, like text, images, or lists. This layout is responsive and will stack vertically on smaller screens.

Right Column

The right column also uses 6 columns out of 12, so it takes the other 50% of the container. The pink background and white text create a nice contrast.

Feel free to customize the colors or content as you like to make it your own!
